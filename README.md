# Travel Booking Data Analysis (SQL Portfolio)

This project demonstrates a real-world data analysis workflow based on travel booking data from professional experience.

## Business Context

The dataset is based on real booking-level data from a travel agency environment, including customer bookings, acquisition channels, destinations, and product types. It reflects actual business operations and is used to analyze performance and generate insights to support decision-making.

All data has been anonymized and structured for demonstration purposes.

## Dataset

The dataset contains 500 booking records with the following fields:

* customer_id
* booking_id
* booking_date
* channel (Google Ads, WeChat, Direct, OTA)
* destination
* product_type (Hotel, Flight, Package, Tour)
* booking_value

## Key Analyses

### 1. Channel Performance Analysis

Evaluates which acquisition channels drive the highest revenue and booking volume, and compares their efficiency.

### 2. Destination Analysis

Identifies top-performing destinations based on revenue, demand, and average booking value.

### 3. Product Analysis

Analyzes booking patterns and revenue across different travel product types to understand value distribution.
